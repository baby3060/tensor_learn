### 텐서플로우 설치 시 에러 떠서 구글링 하여 해결

1. python -m ensurepip : pip 업그레이드 실패 시(pip가 UnInstall 만 되고 Install 안 될 때)
2. python -m pip install --upgrade pip : pip 버전 업그레이드
